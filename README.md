# resumo-lab-AZ900
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO

Durante o laboratório do curso AZ-900 da DIO, a instrutora apresentou o portal do Azure e demonstrou a navegação pelos principais serviços da plataforma. Foram destacados especialmente os recursos mais cobrados no exame, como serviços de computação (máquinas virtuais, Azure Functions e App Services), rede (VNet, VPN, DNS e balanceadores de carga), armazenamento (Blob Storage, File Storage e contas de armazenamento) e segurança (Azure AD, IAM, políticas de acesso e ferramentas de monitoramento). O laboratório teve como objetivo familiarizar o aluno com a interface do Azure e mostrar, na prática, onde localizar e como visualizar os principais serviços fundamentais para a certificação.
